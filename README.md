Healthcare Analytics Dashboard (Power BI)

End-to-End Power BI Project | Real-world Healthcare Use Case

Project Overview

This dashboard analyzes healthcare data to provide insights into patient demographics, cost trends, and payer performance.

Key Features

* Patient Demographics Analysis
* Healthcare Cost Trend Analysis
* Payer Performance Evaluation
* Cost vs Patient Volume (Efficiency Analysis)
* KPI Tracking (Total Cost, Avg Cost, YoY Growth)

Key Insights

* No Insurance is the top contributing payer
* Senior patients have higher healthcare utilization
* Healthcare costs show an increasing trend over time
* Significant variation exists in cost per visit across payers

Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization

Business Value

Helps stakeholders:

* Identify cost drivers
* Improve payer efficiency
* Make data-driven decisions
