Healthcare Analytics Dashboard (Power BI)

End-to-End Power BI Project | Healthcare Analytics DashboardDesigned with a focus on real-world healthcare business decision-making.

Project Overview

This dashboard analyzes healthcare data to provide insights into patient demographics, cost trends, and payer performance.

Key Features

* Patient Demographics Analysis
* Healthcare Cost Trend Analysis
* Payer Performance Evaluation
* Cost vs Patient Volume (Efficiency Analysis)
* KPI Tracking (Total Cost, Avg Cost, YoY Growth)

Key Insights

* No Insurance is the highest cost contributor, indicating potential gaps in coverage
* Senior patients drive higher healthcare utilization
* Significant variation in cost per visit highlights payer inefficiencies
* Cost trends show a steady increase over time

Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization

Data Model
This project follows a star schema:
*Fact Table: Encounters
*Dimension Tables: Patients, Payers, Date
Relationships enable dynamic filtering and efficient analysis across visuals.

Technical Implementation
DAX Measures:
  * Total Cost
  * Cost per Visit
  * YoY Growth %
  * Dynamic KPI Indicators
  * Insight-based text measures

 Features:
  * Interactive slicers
  * Drill-down analysis
  * KPI cards

Business Impact

This dashboard helps stakeholders:
* Identify cost optimization opportunities
* Improve payer efficiency strategies
* Monitor healthcare trends
* Support data-driven decision-making
